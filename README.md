# Beluga Fit

Beluga Fit is a mobile fitness app built with Expo / React Native and backed by Supabase.

## Current focus
- Curated workout plans
- Active workout flow
- Fitness tracking and progress
- Launch hardening and production readiness

## Core flows
- Authentication and session restore
- Browse/select workout plans
- Start workout and complete exercises
- Track user profile and fitness-related data

## Tech stack
- Expo / React Native
- TypeScript
- Supabase
