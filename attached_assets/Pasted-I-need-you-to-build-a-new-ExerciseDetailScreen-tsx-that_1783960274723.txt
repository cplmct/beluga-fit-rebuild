supabase

Query 3 — Session history:
supabase
  .from('session_exercises')
  .select(`
    id,
    workout_sessions (
      id,
      started_at,
      status
    ),
    session_sets (
      set_number,
      weight_kg,
      reps,
      is_completed,
      is_pr
    )
  `)
  .eq('exercise_id', exerciseId)
